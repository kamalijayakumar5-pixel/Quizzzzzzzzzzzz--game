# Quiz Game

A simple browser-based quiz game made using HTML, CSS, and JavaScript.

## Features

- Multiple choice questions
- Score tracking
- Correct and incorrect answer highlighting
- Restart system
- Clean responsive UI
- Instant feedback after answering

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project files
2. Open the folder in VS Code
3. Open `index.html` in your browser

OR

Use the Live Server extension in VS Code:
- Right click `index.html`
- Press "Open With Live Server"
